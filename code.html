// logic function
function reverseString(userString){


    let revString = [];


    //reverse a string using a for loop
    for (let index = userString.length - 1; index >= 0; index--) {
        revString += userString[index];
        

    }
    return revString;

}

Project Documentation: String Reverser

Overview

This project is a JavaScript application that takes a user-entered string from an HTML input element, reverses the string and displays the reversed result on the webpage. It uses three main functions: a controller function (getValues), a logic function (reverseString), and a view function (displayString). The application also toggles the visibility of an alert box to display the result.

Functions

Function 1 : getValue()

Purpose: Acts as a controller function to initiate the string reversal process by retrieving the user's input and coordinating the logic and view functions
Functionality:
1. Hides the alert box by adding the invisible class to the element with id="alert"
2. Retrieves the user's input from the HTML input element with id="UserString"
3. Calls reverseString() to reverse the input string and stores the result in revString
4. Calls displayString() to display the reversed string on the page.
* Parameters: None
* Returns: None

Function 2 : reverseString(userString)
Purpose: Serves as a logic function to reverse the characters of a given string using a for-loop approach
Functionality:
1. Initializes an empty array revString
2. Iterates over userString from the last character to the first, appending each character to revString
3. Returns the reversed string
* Parameters: userString(string): the input string to be reversed
* Returns: revString(string)- The reversed version of userString
* Notes:
1. The function concatenates characters to an array using +=, which implicitly converts the array to a string due to JavaScript's type coercion.

Function 3 : displayString(revString)

Purpose: Serves as the view function to display the reversed characters on the webpage and make the box visible.
Functionality:
1. Updates the inner HTML of the element of the element with id="msg" to show the reversed string in the format: Your word reversed is: ${revString}
2. Removes the invisible class from the element with id="alert", making it visible.
* Parameters:revString(string):The reversed string to display
* Returns: None
